# Pharmarcy management system

This is a web application for pharmarcy management system built using the JAVA It allows users to place an order and admins can manage users,orders,medicine lists.The application includes features such as  real-time medicine availability, and admin management capabilities. The codebase is clean, well-organized, and easy to understand making it suitable for both learning and production use.We done this project as 2nd Year 1sem group project (SLIIT).
